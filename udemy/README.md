# Udemy 字幕下載

## 說明

此腳本支援下載 [Udemy](https://www.udemy.com/) 課程字幕為 Markdown (`.md`) 文件，並提供下載課程影片的功能，方便用戶離線學習或整理筆記。

---

## 功能介紹

- **字幕下載**：
  - 下載整門課程的字幕，生成多個 `.md` 文件。
  - 下載當前課程的字幕，生成單個 `.md` 文件。
  - 支援多種字幕語言（預設為 `en_US`，可自定義）。

- **影片下載**：
  - 支援下載當前課程的影片，格式為 `.mp4`。

- **主要功能**：
  1. 一鍵下載字幕或影片，方便快捷。
  2. 自動從 Udemy API 獲取數據，無需手動設置。

---

## 安裝方法

1. 安裝 [Tampermonkey](https://www.tampermonkey.net/) 瀏覽器插件。
2. 點擊 [Udemy 字幕下載腳本](https://github.com/mawermeow/tampermonkey-scripts/blob/main/udemy-download-subtitles.user.js) 查看腳本代碼。
3. 複製代碼並在 Tampermonkey 新增腳本後貼上。
4. 保存並啟用腳本。
5. 登入 Udemy 並進入課程頁面，測試下載功能。

---

## 測試環境
  - macOS Sequoia 15.2
  - Brave v1.73.104
  - Tampermonkey v5.3.3

---

## 注意事項

1. 若 Udemy API 發生改動，腳本可能無法正常工作。
2. 建議使用最新版本的瀏覽器與 Tampermonkey 確保最佳兼容性。
